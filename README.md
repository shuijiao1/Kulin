# Kulin

Kulin 是基于 [Nezha](https://github.com/nezhahq/nezha) fork 的轻量服务器探针面板，目标是只保留最基础、最常用的服务器监控能力，去掉偏运维平台化的复杂功能。

## 计划保留

### 核心监控

- 服务器列表
- Agent 接入与管理
- 实时状态：在线、CPU、内存、硬盘、负载、网络上下行
- 系统信息：系统、架构、内核、虚拟化、启动时间
- 流量统计：实时流量、总上传/下载、周期流量
- 历史指标图表：CPU、内存、网络、延迟等时间序列
- GeoIP / 位置展示

> 不保留服务器分组。

### 服务监控

- HTTP 监控
- TCP 端口监控
- Ping 延迟监控

### 告警通知

- Telegram 通知渠道
- 告警规则：CPU、内存、硬盘、负载、流量、离线等

### 账号和后台

- 单管理员账号登录
- 基础后台设置

## 计划移除

- 服务器分组
- SSL 证书监控
- 服务可用率复杂统计 / 历史扩展
- 邮件、企业微信等非 Telegram 通知渠道
- 通知分组
- 服务监控告警以外的复杂通知编排
- WebSSH / 网页终端
- 在线文件管理
- 批量执行任务
- 定时任务 / Cron
- Agent 强制更新
- 服务器迁移 / transfer
- MCP 接口 / 文件传输相关能力
- DDNS
- NAT / 端口映射
- 公开 API / API Token / 权限范围
- 多用户、OAuth2、角色权限、在线用户管理
- WAF / 封禁地址
- 多语言 / 多前台主题模板
